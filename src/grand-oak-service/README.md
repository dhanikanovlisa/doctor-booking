# Steps:

### Instalasi and run server
instal node module
```bash
npm install
```
run service with command 
```bash
npm start
``` 
or
```bash
npm run dev
```

### Routers

- Home: http://localhost:3001/
- Slots booking: http://localhost:3001/api/available-slots
- Booking: POST ke http://localhost:3001/api/book
